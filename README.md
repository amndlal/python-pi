# Aura Dashboard

A personal dashboard web app built with Python + Streamlit.

## Features

- Live weather (Cologne)
- Real-time news headlines from Google News
- To-do list (saved locally)
- Quick notes (auto-saved)
- Topic filter: world, technology, business, sports, india, germany

## How to Run

```bash
pip install streamlit requests
python -m streamlit run app.py
```

Then open **http://localhost:8501** in your browser.

## Tech Stack

- Python
- Streamlit
- Google News RSS (no API key needed)
- wttr.in (free weather API)
